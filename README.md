# 🎯 CAT Preparation Dashboard
A personal CAT exam mentor that tracks your preparation, identifies weaknesses,
suggests daily study plans, and predicts your exam readiness timeline.
## Features
- 📝 **Natural Language Logging** — Just describe what you studied in plain English
- 🤖 **AI Parsing** — GPT-4o-mini extracts structured data from logs
- 📊 **Interactive Dashboard** — 6-tab dashboard with charts, progress bars, and metrics
- 🏆 **Readiness Score** — 0–100 score with projected weeks to exam readiness
- 🧠 **AI Coach** — Personalized daily plans, priority topics, and bottleneck analysis
- 📚 **Full CAT Coverage** — QA, DILR, VARC with all topics and subtopics


## Tech Stack
- **Frontend** — Streamlit
- **Database** — Supabase (PostgreSQL)
- **AI** — OpenAI GPT-4o-mini
- **Charts** — Plotly
